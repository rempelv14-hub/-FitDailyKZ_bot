# FitDaily Railway Ready

## Что внутри
- main.py
- requirements.txt
- Procfile
- railway.json
- .env.example

## Как залить на Railway
1. Создай новый проект на Railway
2. Загрузи эти файлы в репозиторий
3. В Variables добавь:
   - BOT_TOKEN
   - ADMIN_TELEGRAM_ID=6954213997
4. Deploy

## Локальный запуск
```bash
pip install -r requirements.txt
python main.py
```
